Github+PicGo搭建免费图床

Created2020-03-20updated2020-03-20Visited

封面图片为TRHX原创

本文参照原创文章：Github+jsDelivr+PicGo 打造稳定快速、高效免费图床

我来说一下自己的使用原由：一开始传图片是从本地和图片地址两种方法，但是图片再次想要找到的话不现实故想办法使用网上的图床

如SM.MS图床,但是仍然是图片没办法整理在一块，最后找到了打造免费图床的方法，也是从hexo主题stun作者的友链要求CDN链接受到的启发，才发现这样的管理图片的方法
首先创建新仓库，然后生成token码，然后下载PicGo配置好参数就能开始用了。
下面是图片测试：苹果高清壁纸（14MB）、

高清壁纸（1MB）（github图床）（SM.MS图床）

高清壁纸42（jsdelivr）（原地址）（SM.MS图床）

FAQ：

Q1：PicGo上传图片失败

A1：仔细检查自定义域名是否与仓库名关系不对应

A2：图片文件名称不能有‘-’或其他特殊字符，如果上传失败，试着修改文件名

A3：软件重启、电脑重启
